# Auto Flow FE

Simple frontend app built with React, TypeScript, Vite, and Tailwind CSS.

## Requirements

-   Node.js 20+
-   npm

## Installation

```bash
npm install
```

## Run the project

```bash
npm run start
```

The app will be available at http://localhost:5173.

## Production build

```bash
npm run build
```

## Preview the build

```bash
npm run preview
```

## Lint

```bash
npm run lint
```

## Storybook

```bash
npm run storybook
```

Build Storybook:

```bash
npm run build-storybook
```

## Tech stack

-   React 19
-   TypeScript
-   Vite
-   Tailwind CSS 4
-   Storybook
